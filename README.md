# COMP90024 - Project2
# Group 49

This is a Twitter analysis system built for COMP90024 at The University of Melbourne.

## Harvester

The Harvester is a python-based Twitter harvester.

## Deployment

The Deployment folder contains Ansible playbooks describing the deployment of the system on the Unimelb Research Cloud however this could easily be adapted to run on any cloud provider.

## Frontend

The frontend intent is to provide a one-page application to visualise the insights drawn from the system.
